# Accessibility Statement for Robotics Textbook

## Our Commitment

We are committed to ensuring digital accessibility for all users, including those with disabilities. We continuously improve the user experience for everyone and apply the relevant accessibility standards.

## Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. We conform to WCAG 2.1 Level AA standards.

## Features for Accessibility

### Keyboard Navigation
- All interactive elements can be accessed using keyboard navigation
- Visual focus indicators are provided for keyboard users
- Logical tab order is maintained throughout the site

### Screen Reader Compatibility
- Proper heading structure (H1, H2, H3, etc.) for content organization
- Alternative text provided for all meaningful images
- ARIA labels and descriptions for interactive components
- Semantic HTML elements used appropriately

### Visual Accessibility
- Sufficient color contrast between text and backgrounds
- Text scaling support up to 200% without loss of functionality
- Clear visual indicators for interactive elements

### Code Examples Accessibility
- Code examples include descriptive titles and captions
- Syntax highlighting with accessible color schemes
- Keyboard-accessible code editors

## Feedback

We welcome feedback on the accessibility of this textbook. If you encounter accessibility barriers or have suggestions for improvement, please contact us at accessibility@example.com.
