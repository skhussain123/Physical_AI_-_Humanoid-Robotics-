# Physical AI & Humanoid Robotics Textbook

Welcome to the comprehensive AI-Native interactive textbook for Physical AI and Humanoid Robotics! This textbook provides a hands-on learning experience with embedded code execution environments for robotics frameworks.

## What You'll Learn

This textbook covers four core modules:

1. **ROS 2 Fundamentals** - The robotic nervous system and communication framework
2. **Gazebo/Unity Simulation** - Physics-based simulation environments for robotics
3. **NVIDIA Isaac** - AI-powered robotics development platform
4. **AI-Driven Robotics** - Machine learning and AI techniques for robotics applications

## Interactive Learning

Each chapter includes:
- Interactive code examples you can run directly in your browser
- Progress tracking to monitor your learning journey
- Hands-on exercises with immediate feedback
- Simulation visualization for robotics concepts

## Getting Started

Begin with the [ROS 2 Fundamentals](./module1-ros2/chapter1.md) module to start your journey in robotics programming.
